Deluge Remote Modern - v

Configuration

This extension requires an accessible version of Deluge web interface to be running. Please refer to the Deluge documentation on how to get this working.

In order to connect to the web interface we need to know some details. The address used to access the interface (https://address:port/) and the password to login is the base level required.

Originally created by YodaDaCoda. Modernized with MV3 and dark mode support.
